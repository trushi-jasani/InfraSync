# InfraSync  
**Unified Control for Buildings and Facilities**

InfraSync is a full-stack **facility management platform** that centralizes building operations including maintenance, billing, amenities, and assets into one secure, data-driven system.

---

##  What it Does
- Manages multiple buildings and users
- Tracks maintenance and work orders
- Handles billing and amenity bookings
- Provides role-based secure access
- Stores all facility data in a structured database

---

##  Tech Stack
**React 路 Node.js 路 Express 路 PostgreSQL 路 JWT 路 REST APIs**

---

##  Why InfraSync?
Most buildings rely on spreadsheets, WhatsApp, and paperwork.  
InfraSync replaces them with a **centralized, automated, and scalable digital platform**.

---

##  Built For
Residential societies, corporate offices, and commercial buildings.
